WEB SEGURA ADAU

XML Despacho:

<DAE-ADAU>
      <Documento>219999820013</Documento>
      <Intercambio>WS_TELEDES</Intercambio>
      <Transaccion>12345678</Transaccion>
      <Fecha>17/10/2014T10:30:25</Fecha>
      <TipoEnvio>N<TipoEnvio>
      <WS>/LuciaWs/awsteledes.aspx</WS>
      <Envio>...xml firmado y codificado en base64...</Envio>
</DAE-ADAU>

Nota: Para los casos que el WS de aduana no requiera un número de transacción, el tag debe venir igual en el XML vacío.

TipoEnvio: Los valores posibles para este tag son:

  • N: Normal, indica al sistema que el envío va contra la URI de producción de Aduanas
  • P: Prueba, indica al sistema que el el envío va contra la URI de pruebas de Aduanas

WS: Los valores posibles para este tag son: (Esta lista puede variar)

  • /LuciaWs/aws_menstock.aspx
  • /LuciaWs/awsarchivadosobre.aspx
  • /LuciaWs/aWSCertOrigen.aspx
  • /LuciaWs/aws_mencor.aspx
  • /LuciaWs/aws_MensManifiesto.aspx
  • /LuciaWs/awsmensimp.aspx
  • /LuciaWs/awspedidocanal.aspx
  • /LuciaWs/awsRespuestaReliquidacion.aspx
  • /LuciaWs/awsteledes.aspx
  • /LuciaWs/awsTramitesDespachante.aspx
  • /LuciaWs/awsviajes.aspx

Nota: El código y descripciones de error pueden ser:

  • 0: Procedimiento Correcto
  • EV01: ERROR INTERNO. PROBLEMAS DE BASE DE DATOS O SISTEMA
  • EV02: ERROR: UD NO ESTA AUTORIZADO A CONSULTAR LA TRANSACCION SOLICITADA
  • EX01: ERROR: XML RECIBIDO NO TIENE FORMATO VALIDO
  • ES01: ERROR: FIRMA DIGITAL MAL FORMADA
  • ES02: ERROR: INTEGRIDAD DEL MENSAJE COMPROMETIDA
  • ES03: ERROR: FIRMA DIGITAL VENCIDA
  • ES04: ERROR: FIRMA DIGITAL NO AUTORIZADA POR ADAU
  • EA001: ERROR: EL SERVICIO DE ADUANA ESTA CAIDO, CONSULTE EN UNOS MINUTOS
  • EA002: ADUANA NO RESPONDIO EN EL MOMENTO, CONSULTE EN UNOS MINUTOS
